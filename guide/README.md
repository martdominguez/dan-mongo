# Estructura de modulos

Cada modulo del curso debe seguir una estructura repetible y facil de mantener.

## Convencion

Cada carpeta de modulo incluye:

- `README.md`: alcance del modulo y objetivos de escritura
- `01-teoria.md`: explicacion conceptual progresiva
- `02-detalle-tecnico.md`: comandos, operadores, sintaxis y matices
- `03-ejemplos-practicos.md`: ejemplos guiados con datos realistas
- `04-ejercicios.md`: consignas breves para practicar
- `05-repaso.md`: preguntas de revision y chequeo de comprension

## Principio editorial

En esta primera etapa, la estructura queda preparada para desarrollar contenido despues. La prioridad es consistencia de autoria, no completar todo el curso de una vez.
